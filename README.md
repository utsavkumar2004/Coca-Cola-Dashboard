# 🥤 Coca-Cola Sales Analytics Dashboard

## 📌 Project Overview

A professional **Power BI sales dashboard** built to analyze Coca-Cola sales performance across the United States.

The dashboard provides deep insights into revenue generation, operating profit, beverage performance, and state-wise sales trends to support strategic business decision-making.

---

## 🚀 Objectives

✔ Monitor sales performance
✔ Analyze operating profit trends
✔ Compare beverage brand performance
✔ Track unit sales
✔ Visualize state-wise distribution
✔ Identify key profit influencers

---

## 📊 Dashboard Highlights

### 🔹 Key Metrics

* 💰 Total Sales
* 📦 Units Sold
* 📈 Operating Profit
* 💵 Average Price
* 🗓 Date Range Analysis

---

## 📈 Visual Reports Included

### 🏆 Brand-wise Sales Analysis

Compare performance across beverage brands.

### 📍 Sales by US States

Interactive geographic bubble map for regional analysis.

### 📊 Operating Profit Analysis

Understand major factors influencing profitability.

### 🔍 Key Influencer Analysis

AI-powered insights for operational decision-making.

### 📅 Date-wise Filtering

Dynamic trend exploration over selected periods.

✨ Coca-Cola leads in overall revenue generation
✨ Regional sales distribution highlights high-performing states
✨ Certain brands significantly outperform others
